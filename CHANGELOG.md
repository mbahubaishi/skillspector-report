# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - 2026-07-08

Initial public release.

### Added

- Local PDF report generation from NVIDIA SkillSpector JSON output
- Executive summary, findings detail, scanned components, raw JSON appendix, and legal notice
- Optional logo and report metadata fields
- Static-only SkillSpector scan mode using `--no-llm`
- Safe and malicious example PDF reports
- GitHub Security Checks workflow
- OpenSSF Scorecard workflow
- MIT license
- Security policy
