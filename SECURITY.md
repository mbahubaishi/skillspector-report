# Security Policy

## Reporting Security Issues

If you believe you have found a security issue in SkillSpector Report, please do not open a public issue with sensitive details.

Instead, contact the maintainer through GitHub:

- [@mbahubaishi](https://github.com/mbahubaishi)

This policy applies to security issues in this report generator, such as unsafe command execution, unexpected local data leakage, or unsafe handling of generated reports.

Findings reported by NVIDIA SkillSpector about scanned third-party skills, packages, folders, repositories, or zip files are not vulnerabilities in this project.

SkillSpector Report is an independent companion tool for NVIDIA SkillSpector and does not guarantee that scanned content is safe.
